This folder contains the Stata file 'Election Data' and the Excel file 'pewissues', as well as the Stata do-file 'Election do-file' used in the analysis for the paper.

'Election Data' contains data on US Presidential Elections from 2000-16 on a county level, from Dave Leip's US Election Atlas,
as well as economic and demographic county characteristics from multiple years of the US Census Bureaus' American Community Survey
(ACS) as well as the Bureau's County Business Patterns (CBP) dataset. It contains additional data on religious adherence from the Association of Religion Data Archives (ARDA).

'pewissues' survey response data on voters priorities' in the 2012 and 2016 Presidential Elections,
for which the sources are: http://www.people-press.org/2016/07/07/4-top-voting-issues-in-2016-election/ and http://www.people-press.org/2012/09/24/for-voters-its-still-the-economy/

The Stata do-file 'Election do-file' contains the code for the statistical analysis, and produces all tables and charts contained in the paper.
